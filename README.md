# Security Risk Assessment & Compliance Mapping Tool

This project is a Python-based security risk assessment tool that evaluates system security controls using a quantitative risk model(Likelihood × Impact) and maps compliance gaps to ISO 27001 and NIST frameworks.


## Features
- CLI-based assessment tool
- Web-based risk assessment using Flask
- Quantitative risk scoring
- ISO 27001 & NIST SP 800-53 mapping
- Risk level classification (Low / Medium / High)
- Identification of failed security controls
- Automatic Audit-style risk report generation
- Likelihood × Impact risk scoring

## Technologies Used / ## Tech Stack
- Python
- Flask
- CSV
- HTML

## How to Run (CLI)
```bash
python main.py
```

## How to Run Locally
```bash
pip install -r requirements.txt
python app.py
```
